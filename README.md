Referral Management System (MERN Stack Backend)
This project implements a robust backend system for user registration, authentication, and a credit-based referral program using Node.js, Express, and Mongoose. The core feature is accurately tracking referral relationships and conditionally awarding credits upon a referred user's first purchase.

✨ Features
Secure Authentication: User registration and login with bcryptjs for secure password hashing.

Unique Referrals: Every user receives a unique referralCode upon registration.

Referral Tracking: Records the referrer-referred relationship in a dedicated Referral model.
